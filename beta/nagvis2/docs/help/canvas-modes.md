# Canvas-Formate verstehen

NagVis 2 bietet vier verschiedene Modi für dein Dashboard:

1. **Frei (Free)**: Das Canvas passt sich immer zu 100% deinem Browserfenster an. Ideal für flexible Dashboards.
2. **Seitenverhältnis (Ratio)**: Behält ein festes Format (z.B. 16:9). Perfekt für Kontrollraum-Monitore.
3. **Feste Auflösung (Fixed)**: Definiert Pixel-genau (z.B. 1920x1080). Zoomt automatisch.
4. **Hintergrund (Background)**: Die Map ist exakt so groß wie dein hochgeladenes Bild.
