# ZIP-Import & Export

Mit dem ZIP-Format kannst du komplette Maps inklusive aller Einstellungen und Hintergrundbilder sichern oder mit anderen teilen.

### Export
Klicke im Map-Menü auf **Exportieren**, um ein Bundle zu erstellen. Dieses enthält:
* Die Map-Konfiguration (`map.json`)
* Das Hintergrundbild (falls vorhanden)

### Import
Beim Import kannst du eine neue **Map-ID** vergeben. Falls bereits eine Map mit derselben ID existiert, wird diese nur überschrieben, wenn du den Import bestätigst.
